Daniel Giguere

MSc. Candidate

My current project involves creating an exploratory data visualization app using the ‘Shiny’ R package. The goal of this project is to create a simple graphical user interface allowing users to easily create PCA biplots (and other plots) to explore their –omic data. The app also allows users to filter and colour their biplots according to metadata for further exploration of multivariate data.
